# Simon-Game
Simon is a memory game where a sequence of patterns is generated and user is required to repeat the sequence. The game is made by implementing jQuery.
Open the index.html file to start with the game.
In order to start the game, press any key from the keyboard and repeat the patterns generated by the colored buttons with each level.
If one fails to generate the same pattern, Game over message is displayed and one can start over by pressing any key from the keyboard.
